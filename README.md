
# **opFishing**‎ - Over Power Loot

Make fishing OP again! This addon replaces boring fish and junk with powerful loot like enchanted books, netherite, elytra, and even totems. It's like opening a gacha with every cast!

Every cast matters.

The drop table is carefully tuned to:

*   Keep common rewards useful
*   Make valuable drops feel exciting
*   Preserve rare items as true jackpots

This is not a chaotic loot spam datapack. ‎It is a balanced endgame enhancement built for players who want meaningful rewards without breaking survival.

## **Loot List**

Percentages are approximate and based on weighted distribution.

### Common Rewards (~38%)

*   Enchanted Book — ~6.4%
*   Iron Ingot (5–15) — ~7.3%
*   Gold Ingot — ~6.4%
*   Diamond (5–15) — ~4.6%
*   Firework Rockets — ~6.4%
*   Experience Bottles — ~6.4%

### Valuable Rewards (~21%)

*   Netherite Upgrade Smithing Template — ~4.6%
*   Ender Eye — ~3.7%
*   Shulker Shell — ~3.7%
*   Gold Block (1–3) — ~2.8%
*   Iron Block (1–3) — ~2.8%
*   Diamond Block — ~1.8%
*   Netherite Ingot — ~1.8%

### Rare Core Drops (~7%)

*   Totem of Undying — ~2.8%
*   Elytra — ~1.8%
*   Enchanted Golden Apple — ~1.8%
*   End Portal Frame — ~0.9%

### God Armor (~7%)

Each piece drops fully enchanted.

*   Netherite Helmet — ~1.8%
*   Netherite Chestplate — ~1.8%
*   Netherite Leggings — — ~1.8%
*   Netherite Boots — ~1.8%

### God Weapons & Tools (~20%)

Each item drops fully enchanted.

*   Netherite Sword — ~1.8%
*   Fortune Pickaxe — ~1.8%
*   Silk Touch Pickaxe — ~1.8%
*   Netherite Axe — ~1.8%
*   Netherite Shovel — ~1.8%
*   Mace — ~1.8%
*   Bow — ~1.8%
